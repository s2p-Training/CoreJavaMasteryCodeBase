## üß© 1. What is an Array?
An **array** is a fixed-size, ordered collection of elements of the same type stored in contiguous memory locations.  
Each element can be accessed using an **index**.

### Key Characteristics
- Homogeneous elements (same data type).
- Zero-based indexing.
- Fixed length once created.
- Random access via `array[index]`.
- `array.length` gives size.

| Syntax Type | Example | Description |
|--------------|----------|-------------|
| Using element list | `int[] nums = {1,2,3,4,5};` | Direct initialization |
| Using `new` keyword | `int[] nums = new int[5];` | Declares empty array with default values |
